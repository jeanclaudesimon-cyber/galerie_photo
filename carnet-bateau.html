✅ Check-list Sécurité avant départ

  • Gilets de sauvetage pour tous à bord
  • Extincteur vérifié et accessible
  • Fusées de détresse valides
  • VHF chargée + test radio
  • Ancre + chaîne + bout de mouillage
  • Pompe de cale fonctionnelle
  • Trousse de secours complète
  • Feux de navigation OK
  • Météo consultée + marées
  • Carburant niveau + réserve
  • Téléphone chargé étanche
  • Prévenir quelqu’un à terre

🪢 Nœuds indispensables

Nœud de chaise Amarrage, ne glisse pas, se défait facilement
Nœud de cabestan Amarrer à un taquet rapidement
Nœud de taquet Bloquer une écoute sur un taquet
Nœud plat Joindre 2 bouts même diamètre
Nœud en huit Nœud d’arrêt en bout d’écoute
Tour mort + 2 demi-clés Amarrer sur un anneau ou une bite
