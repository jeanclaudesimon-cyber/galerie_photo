Comment fonctionne le lien entre CSS en haut et HTML en bas ? Par exemple .info-card h3 et mon <h3> dans le code ?

C'est ce qu'on appelle un sélecteur CSS. Il dit : "Applique ce style à tous les h3 qui se trouvent DANS un élément avec la classe info-card".

/* CSS en haut */ .info-card h3 { color: var(--vert2); font-size: 16px; } /* HTML en bas — le navigateur fait le lien automatiquement */ <div class="info-card"> <h3>💡 Mon titre</h3> ← reçoit le style </div>
Astuce : Le CSS s'applique automatiquement à tous les éléments qui correspondent au sélecteur — pas besoin de répéter le style à chaque fois !
❓

Les balises HTML — comment ça fonctionne ? Ouvrir et fermer ?

La règle est toujours la même : ouvrir puis fermer avec un slash /

<balise>contenu</balise>
BaliseEffetExemple
<strong>Gras<strong>texte</strong>
<em>Italique<em>texte</em>
<p>Paragraphe<p>texte</p>
<h1> à <h6>Titres<h2>Mon titre</h2>
<a>Lien<a href="url">texte</a>
<img>Image<img src="photo.jpg">
<br>Saut de lignepas de fermeture
Erreur classique : </strong>texte</strong> ← le slash ne doit pas être au début ! La bonne version : <strong>texte</strong>

C'est quoi la différence entre <br><br> et des <p> séparés ?

/* Mauvaise pratique — simuler des paragraphes avec br */ <p>Premier texte.<br><br> Deuxième texte.</p> /* Bonne pratique — vrais paragraphes séparés */ <p>Premier texte.</p> <p>Deuxième texte.</p> /* Gérer l'espace dans le CSS */ p { margin-bottom: 1rem; }

Les div imbriquées — comment ne pas se perdre dans les fermetures ?

Chaque <div> ouverte doit être fermée avec </div>. Entre les vidéos on a 2 fermetures, à la toute dernière on en a 3 :

<div class="vid-card"> <div class="vid-embed"> <iframe src="..."></iframe> </div> ← ferme vid-embed <div class="vid-info"> <div class="vid-title">Titre</div> <div class="vid-date">Date</div> </div> ← ferme vid-info </div> ← ferme vid-card /* Dernière vidéo : ajouter encore un </div> pour fermer le conteneur videos */ </div> ← ferme videos (seulement à la fin !)
Astuce pro : Ajouter des commentaires pour s'y retrouver !
</div><!-- fin vid-card --> — invisible sur la page mais très utile !

C'est quoi la différence entre le style inline et le CSS en haut ?

/* Style inline — directement dans la balise HTML */ <p style="font-size:13px;color:red">texte</p> /* CSS en haut — bien mieux ! */ .info-card p { font-size: 13px; color: red; } /* Et dans le HTML simplement : */ <p>texte</p> ← plus besoin du style inline !
Avantage CSS en haut : Si tu veux changer la couleur un jour, tu ne la changes qu'à un seul endroit et ça se met à jour partout automatiquement !
❓

Comment mettre un mot en gras ET en vert avec <strong> ?

/* Option 1 — style inline (rapide mais pas propre) */ <strong style="color:#6aad5e">texte</strong> /* Option 2 — dans le CSS (propre et réutilisable) */ .info-card strong { color: var(--vert2); } .hero-info strong { color: var(--vert2); } /* Option 3 — partout sur la page */ strong { color: var(--vert2); }
❓

C'est quoi les variables CSS comme var(--vert2) ?

Les variables CSS se définissent une fois en haut dans :root et s'utilisent partout avec var(--nom) :

:root { --vert: #4a7c3f; --vert2: #6aad5e; ← vert clair --or: #c8941a; } /* Utilisation */ h3 { color: var(--vert2); } strong { color: var(--vert2); }
Avantage : Si tu veux changer le vert de tout ton site, tu changes juste une ligne dans :root !
❓

C'est quoi le media query et comment l'utiliser pour le responsive mobile ?

Le media query applique des styles différents selon la taille de l'écran :

/* Style normal — PC */ .hero { grid-template-columns: 1fr 1fr; } ← 2 colonnes /* Style mobile — écran < 768px */ @media(max-width:768px) { .hero { grid-template-columns: 1fr; } ← 1 colonne .hero-img img { height: 220px; } ← image moins haute .videos { grid-template-columns: 1fr; } }

Comment activer le mois d'Avril (ou Mai, Juin...) quand j'ai de nouvelles vidéos à ajouter ?

Il faut faire deux choses dans la section du mois concerné :

1. Sur le header — retirer le style="opacity:.5"

2. Sur la div videos — retirer le style="display:none"

/* AVANT — mois inactif */ <div class="mois-header" style="opacity:.5"> <h3>Avril 2026</h3> <span class="mois-count">À venir</span> </div> <div class="videos" style="display:none"> ...vidéos... </div> /* APRÈS — mois actif */ <div class="mois-header"> ← opacity:.5 supprimé <h3>Avril 2026</h3> <span class="mois-count">2 vidéos</span> ← mettre le bon nombre </div> <div class="videos"> ← display:none supprimé ...vidéos... </div>
Astuce : Mets aussi à jour le compteur mois-count avec le bon nombre de vidéos !

Comment ajouter une vidéo dans le mois ? Copie ce bloc et remplace XXXXXXXX par l'ID YouTube :

<div class="vid-card"> <div class="vid-embed"> <iframe src="https://www.youtube.com/embed/XXXXXXXX" allowfullscreen loading="lazy"></iframe> </div> <div class="vid-info"> <div class="vid-title">Titre de la vidéo</div> <div class="vid-date">📅 Avril 2026</div> </div> </div> ← 4 </div> pour fermer : vid-info, vid-card... c'est normal !

/* Structure complète d'un mois inactif à coller après Mai */ <div class="mois-section"> <div class="mois-header" style="opacity:.5"> <span style="font-size:20px">🦅</span> <h3>Juin 2026</h3> <span class="mois-count">À venir</span> </div> <div class="videos" style="display:none"> <!-- VIDÉOS JUIN : coller ici les blocs vid-card --> </div> </div>
Règle d'or : Chaque <div> ouvert doit être fermé par un </div>. Pour ne pas se perdre : mois-section contient mois-header + videos, et videos contient autant de vid-card que tu veux !

Loffer vs Abattre : les deux manœuvres de base pour changer de cap

Loffer et abattre sont les deux manœuvres fondamentales pour changer de cap par rapport au vent. Loffer consiste à rapprocher la proue du bateau de la direction du vent en tirant la barre, ce qui fait gîter et nécessite de border les voiles. Abattre est l'action inverse, poussant la barre pour éloigner le bateau du vent, ce qui nécessite de choquer les voiles.

① Abattre : s’éloigner du vent

Action à la barre : Pousser la barre franche sous le vent ou tourner la barre à roue vers le côté opposé au vent.
Voiles : Il faut choquer = relâcher les voiles pour qu’elles restent gonflées sans faseyer.
Effet : Le bateau se met plus à plat, le vent vient davantage de l’arrière et le bateau accélère au vent portant.

② Loffer : se rapprocher du vent

Action à la barre : Tirer la barre franche vers soi au vent ou tourner la barre à roue vers le vent.
Voiles : Il faut border = tirer les voiles car l’angle avec le vent diminue.
Effet : Le bateau gîte davantage, la vitesse peut diminuer si l’on se rapproche trop du vent debout.

Virer de bord vs Empanner : les deux façons de changer d’amure

Pour changer d’amure, c’est à dire faire passer le vent de l’autre côté du bateau, il existe deux manœuvres : virer de bord et empanner. Virer se fait face au vent et c’est sécurisé. Empanner se fait dos au vent et c’est dangereux à cause de la bôme.

① Virer de bord : on passe FACE au vent

Quand : Tu es au près ou au bon plein et tu veux changer de côté.
Action à la barre : Tu loffes à fond jusqu’à passer le lit du vent. Tu pousses la barre sous le vent.
Voiles : Le foc passe de l’autre côté tout seul. La grand-voile claque un peu puis se regonfle. Tu bordes après.
Commandement : "Paré à virer ?" → "Envoyez !"
Risque : Faible. Si tu rates ta manœuvre, tu te retrouves face au vent et le bateau s’arrête. C’est tout.

② Empanner : on passe DOS au vent

Quand : Tu es au largue ou vent arrière et tu veux changer de côté.
Action à la barre : Tu abats à fond jusqu’à passer vent arrière. Tu pousses la barre pour que la poupe passe par le lit du vent.
Voiles : La grand-voile passe d’un coup sec de l’autre côté. DANGER BÔME !
Commandement : "Paré à empanner ?" → "Attention à la bôme !" → "Envoyez !"
Risque : ÉLEVÉ. La bôme traverse le cockpit très vite. Si quelqu’un a la tête sur la trajectoire, c’est l’accident. On borde toujours la grand-voile au centre avant d’empanner pour limiter le choc.

Moyen mnémotechnique pour pas se tuer

Virer = "V" comme "Vent devant" = tranquille, tu peux virer.
Empanner = "E" comme "Eh merde la bôme !" = dangereux, tu annonces et tu fais gaffe.

En résumé : Tu vires au près. Tu empannes au portant.

⛵

Cas concret : Je suis tribord amure au près, on me dit d’empanner. Je fais quoi ?

Tu es tribord amure au près = vent à tribord, voile à tribord. On te dit d’empanner. STOP ! Tu ne peux pas empanner directement.

Pourquoi tu peux pas empanner direct ?

Empanner = changer d’amure en passant dos au vent.
Problème : Au près, tu as le vent qui vient de l’avant à 35°.
Pour empanner, il faudrait que le vent vienne de l’arrière à 180°.
Conclusion : Il faut d’abord abattre jusqu’au vent arrière avant d’empanner.

La bonne séquence si tu veux vraiment empanner

1. Abattre d’abord : Tu pousses la barre sous le vent. Tu choques la voile au fur et à mesure.
2. Tu passes au travers : Vent à 90°, voile à moitié choquée.
3. Tu passes au largue : Vent à 135°, voile choquée à fond.
4. Tu arrives vent arrière : Vent à 180°, voile perpendiculaire.
5. SEULEMENT LÀ tu empannes : "Paré à empanner ?" → "Attention à la bôme !" → "Envoyez !"
6. Tu te retrouves bâbord amure au largue de l’autre côté.

La vraie question : pourquoi empanner au près ?

Réponse : On empanne JAMAIS au près.
Si tu es au près tribord amure et que tu veux passer bâbord amure, tu VIRES de bord.
Tu loffes, tu passes face au vent, et c’est réglé en 10 secondes sans danger.

Si le skipper te dit "empanne" alors que tu es au près, c’est qu’il se trompe de terme.
Soit il veut dire "vire", soit il veut que tu abattes d’abord pour changer d’allure, puis que tu empannes plus tard.
Dans le doute : Tu demandes "Tu veux qu’on vire ou qu’on abatte d’abord ?"

Règle d’or à retenir

Au près et bon plein → On VIRE de bord
Au largue et vent arrière → On EMPANNE

Si on te demande d’empanner alors que t’es au près, c’est soit une erreur, soit un test pour voir si tu dors.

Anatomie du voilier : les 9 pièces à connaître avant d’embarquer

Voici les éléments essentiels d’un voilier, numérotés sur la photo. Retiens-les, le skipper va les utiliser toute la journée.

① La Bôme

C’est quoi : La grosse barre horizontale en bas de la grand-voile.
À quoi ça sert : Elle tient le bas de la grand-voile et permet de la border ou choquer.
Danger : Lors d’un empannage, elle traverse le cockpit à toute vitesse. BAISSE LA TÊTE !
Commandement : "Attention à la bôme !" = tout le monde se baisse.

② Le Mât

C’est quoi : Le grand poteau vertical au milieu du bateau.
À quoi ça sert : Il porte les voiles. Tout est accroché dessus : grand-voile, génois, drisses.
À savoir : Il est tenu par les haubans sur les côtés et l’étai à l’avant. Si un hauban lâche, le mât tombe.

③ Le Génois

C’est quoi : La voile d’avant, triangulaire, qui recouvre en partie la grand-voile.
À quoi ça sert : C’est le moteur principal au près. Plus grand qu’un foc.
Réglage : On le borde avec l’écoute de génois qui passe dans un winch.
Terme : Quand tu changes d’amure, "le génois passe" de l’autre côté.

④ Le Hale-bas

C’est quoi : Le bout ou le vérin qui tire la bôme vers le bas depuis le pied de mât.
À quoi ça sert : Il empêche la bôme de remonter quand tu choques au portant. Ça aplatit la grand-voile.
Réglage : Au près tu le tends. Au vent arrière tu peux le mollir un peu.
Astuce : Si la bôme remonte toute seule, ton hale-bas est pas assez tendu.

⑤ Le Chariot d’écoute de grand-voile

C’est quoi : Le rail avec une poulie qui coulisse dans le cockpit, sous la bôme.
À quoi ça sert : Il permet de déplacer le point de tire de l’écoute de grand-voile à gauche ou à droite.
Réglage : Au près tu le mets au centre ou au vent. Au portant tu le laisses partir sous le vent.
Terme exact : On dit "chariot d’écoute" ou "barber hauler de grand-voile". C’est ça que t’appelais "poulie de bôme dans cockpit".

⑥ Le Hauban

C’est quoi : Les câbles en acier qui tiennent le mât sur les côtés.
À quoi ça sert : Ils empêchent le mât de tomber à bâbord ou à tribord.
À savoir : Il y a les haubans = côtés, et l’étai = devant, et le pataras = derrière.
Sécurité : Avant de partir, on vérifie qu’ils sont bien tendus et pas effilochés.

⑦ La Barre franche

C’est quoi : Le grand bâton en bois relié directement au safran.
À quoi ça sert : Tu diriges le bateau avec. C’est le volant du voilier.
Règle : Tu pousses la barre du côté où tu veux que l’arrière du bateau aille.
Mémo : Barre à tribord = nez à bâbord. Barre à bâbord = nez à tribord.

⑧ Le Winch

C’est quoi : Le tambour métallique avec une manivelle sur le côté du cockpit.
À quoi ça sert : À démultiplier ta force pour border les écoutes de génois ou de spi.
Comment : Tu fais 3 tours avec l’écoute, tu mets la manivelle, tu tournes dans le sens des aiguilles.
Sécurité : Attention aux doigts. On ne met JAMAIS la main entre l’écoute et le winch.

⑨ Le Lazy jack

C’est quoi : Les petits bouts en V qui relient la bôme au mât.
À quoi ça sert : Quand tu affales la grand-voile, elle tombe toute seule dans les lazy jack comme dans un hamac.
Avantage : T’as pas besoin de 3 équipiers pour ranger la grand-voile. Elle se plie toute seule.
Terme : On dit aussi "paresseux" en français. Lazy jack = "paresseux Jack".

Comment régler les voiles au près par vent de 22 nœuds ?

Au près par 22 nœuds, le vent est soutenu — il faut border et aplatir les voiles pour réduire la gîte et maintenir la vitesse sans perdre le contrôle.

🔺 Grand-voile

Écoute de GV : Border franchement — le chariot d'écoute légèrement sous le vent pour ouvrir le haut de la voile et décharger.
Cunningham : Choquer légèrement pour avancer le creux vers l'avant.
Halebas : Border pour cintrer le mât, aplatir la voile et réduire la gîte.
Balestron/bordure : Légèrement choqué pour ouvrir le tableau arrière.
À 22 nœuds : Si gîte excessive, envisager un ris dans la grand-voile.

🔷 Génois / Foc

Écoute de génois : Border pour fermer le génois — le guidant doit être parallèle à l'axe du bateau.
Chariot d'écoute : Reculer légèrement pour ouvrir le haut du génois et éviter de bloquer le vent sur la GV.
À 22 nœuds : Si le bateau gîte trop, enrouler 20 à 30% du génois pour réduire la toile sans changer de voile.

⚖️ Équilibre du bateau

Ardent : Si le bateau cherche à loffer tout seul — choquer légèrement la GV ou border le génois.
Mou : Si le bateau abat tout seul — border la GV ou choquer le génois.
Barre : À 22 nœuds au près, une légère pression à la barre est normale — si elle devient lourde, réduire la toile.
Vindo 40 : Bateau solide et stable — il supporte bien 22 nœuds au près avec un ris si nécessaire.

Règle d'or au près : On borde pour accélérer, on choque pour réduire la gîte. À 22 nœuds, la priorité c'est de garder le bateau plat et équilibré plutôt que de chercher la vitesse maximale !

7.6 kts en pointe — le Vindo 40 exprime bien son potentiel au bon plein, allure où il est à l'aise.

Le ris génois à 23 nœuds au bon plein était la bonne décision — ça évite de surtoiler et garde le bateau équilibré sans sacrifier trop de vitesse.

📐 Bon plein — rappel de l'allure

Angle : Entre 60° et 90° du vent — plus ouvert que le près, moins que le travers.
Voiles : Légèrement moins bordées qu'au près — le creux des voiles travaille davantage.
Comportement : Allure rapide et confortable — la gîte est moins prononcée qu'au près, la barre plus légère.
Avantage : C'est souvent l'allure la plus efficace pour un voilier de croisière comme le Vindo 40.

À retenir : Par 23 nœuds au bon plein, un ris GV + génois réduit = bateau équilibré, barre légère et 6.5 kts de moyenne. La réduction de toile n'a pas pénalisé la vitesse — elle a optimisé le rendement ! ⛵

Bâbord, Tribord, Avant, Arrière... Comment s'y retrouver ?

Sur un voilier, on ne dit jamais "gauche" ou "droite" car cela dépend d'où l'on regarde. On se place toujours face à l'avant (la proue).

Babord : Gauche Tribord : Droite Proue : L'avant du bateau
Poupe : L'arrière du bateau

💡 Moyen mnémotechnique : Utilisez le mot "BATTERIE" :

• BA (Bâbord) est à gauche.

• TRIE (Tribord) est à droite.

(Ou le mot "PI-CE" : bâbord a 2 syllabes comme "gauche", tribord a 2 syllabes comme "droite"... mais Batterie est infaillible !)

Pourquoi mon onduleur décroche quand il fait chaud ?

Tension réseau > 253V. Norme = 230V ±10%. Si le quartier injecte à fond (voisin 34 panneaux), la tension grimpe → l’onduleur coupe pour protéger le réseau et ton matos.

Symptôme : 24° + soleil 13h-15h = pic d’injection → *clic* décrochage. Tu perds la prod jusqu’à ce que la tension rebaisse
❓

Comment j’ai réduit mes décrochages alors que les voisins trinquent ?

Test des 3 phases. J’ai mesuré au multimètre entre L1-N, L2-N, L3-N en pleine journée. Il y avait 2-3V d’écart.

J’ai branché l’onduleur sur la phase la moins haute. Résultat : je décroche beaucoup moins que le voisin 34 panneaux qui est sur la phase saturée.

// Exemple relevé midi solaire
L1-N = 251.8V ← saturée
L2-N = 249.2V ← OK
L3-N = 250.5V ← limite
→ Branchement sur L2
Astuce : Ne dis rien aux voisins 😉 Si tout le monde bascule sur la bonne phase, elle sature aussi et plus personne ne produit
